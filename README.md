# clippy_maker_gui
python ffmpeg video clipping GUI because I'm tired of messing up in the command line and "I'm a visual person" who wanted to experience the pain of coding a GUI.

# installation
from a terminal ```pip install xxx```

- tkinter
- RangeSlider
